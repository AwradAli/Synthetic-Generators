Synthetic Generator for Multi-Links Generator

This repository includes the synthetic network generator (MLG) code for our paper: Synthetic Generators for Cloning Social Network Data. Awrad Mohammed Ali, Hamid Alvari, Nima Hajibagheri, Kiran Lakkaraju, and Gita Sukthankar.

1)"synthetic_generator.m" file contains the main code for generating MLG network

2)"eventChooser.m" file contains the code for choosing between different links types

3)"frequency.txt" file contains an example of the links frequency input.

4)"relations.txt" file contains an example of the co-occurance matrix input

5)An example of how to run the generator is explained in "input.txt" file.
